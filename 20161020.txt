[R]
resources

[Activity]
뭔가를 불러오고 실행하고, 화면에는 띄워주는 상호작용

[View]
화면에 내용을 그리는 구조체

[Manifest]
앱에 대한 정보를 기록

앱의 보유 액티비티 수, 사용 하드웨어 수, 권한

사용이유 -> 속도때문 -> 메인 액티비티 위치, 사용 가능 권한등에 대한 정보를 보유하고 시스템에 알려줌으로써, 시스템에 불필요한 동작을 줄임


[권한]

[HTTP]
hypertext Transfer Protocol
www를 통하여 정보 교환

네이버 웹이 소켓통신안하고 http로만 통신

다 http쓰는데 소켓통신의 장점이 뭐냐

내가 필요없는데 쓸모없는정보가 날아간다 -> 손실발생

http안에서 문서를 요청하는 방법에는 여러가지가 있다


http는 통신에 xml을 이용, json을 이용

xml : 데이터 교환 및 그 외 목적으로 다목적 사용, json에 비해 텍스트가 많이 포함
json : 텍스트가 적게 들어가며 javascript과의 호환

xml이 json보다 용량이 커서 시스템이 텍스트를 해석할때 json이 유리


[gson from string]
json을 자바 객체 처럼 사용가능하게 변경해주는 기능 gson
